# User Authentication

### Techstack
1. NestJs
2. ReactJs
3. MongoDB
4. Swagger

## Backend

### Run
```bash
cd backend
npm install
npm run start:prod
```

### Open API (Swagger Docs)
PORT depends on the one configured in env file
```bash
http://localhost:4000/api 
```

## Frontend

### Run
```bash
cd frontend
npm install
npm run start
```
